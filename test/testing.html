Pause Button

0x00E0

  • Clears The Screen

0x00EE

  • SUBTRACTS 1 FROM SP
  • PC SET TO TOP OF STACK

0x1NNN

  • Jumps to Address

0x2NNN

  • Calls subroutine
  • Stores PC on Stack

0x3NNN

  • Skips Instruction if VX == NN
  • Doesn't Skip if VX != NN

0x4NNN

  • Skips Instruction if VX != NN
  • Doesn't Skip if VX == NN

0x5XYN

  • Skips Instruction if VX == VY
  • Doesn't Skip if VX != VY

0x6XNN

  • Sets VX to NN
  • Updates PC

0x7XNN

  • Stores VX + NN
  • Updates PC

0x8XY0

  • Sets VX to VY
  • Updates PC

0x8XY1

  • Stores VX OR VY
  • Updates PC

0x8XY2

  • Stores VX AND VY
  • Updates PC

0x8XY3

  • Stores VX XOR VY
  • Updates PC

0x8XY4

  • ADD VX, VY
  • SETS VF Flag Properly
  • Updates PC

0x8XY5

  • SUB VX, VY
  • SETS VF Flag Properly
  • Updates PC

0x8XY6

  • SHR VX, 1
  • SETS VF Flag Properly
  • Updates PC

0x8XY7

  • SUBN VX, VY
  • SETS VF Flag Properly
  • Updates PC

0x8XYE

  • SHL VX, 1
  • SETS VF Flag Properly
  • Updates PC

0x9XY0

  • SKIP IF VX != VY
  • Updates PC

0xANNN

  • SET I TO NNN

0xBNNN
